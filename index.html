User: "Link expired" / embedded signing blank / signer stuck on one field.
0/5

  2. 2Internal admin profile: account state, request status, expiry, routing order, field placement

2
Sign · Webhook / API integration
Developer's CRM stuck "pending" — signature completed but webhook never fired.
0/5
  1. 1Confirm signature_request ID and event type expected (signed, all_signed)
  2. 2Admin profile + webhook delivery log — HTTP status, retries, response body
  3. 3Sentry + Datadog: handler error vs customer endpoint — APM trace on API callback path
  4. 4Validate callback URL, TLS cert, customer's endpoint returns 2xx — test API key / OAuth token
  5. 5Remediate with customer or Jira to eng — event IDs, Sentry/Datadog links, payload snapshot
Result: Integration synced. Customer workflow resumes.
3
Fax · Sent but not received
User: fax shows delivered in Dropbox — recipient says nothing arrived.
0/5
  1. 1Get fax ID, sender account, recipient number, and timestamp sent
  2. 2Internal admin profile: verify number, account entitlement, sender permissions
  3. 3Transmission report in admin profile — failed vs delivered; Sentry/Datadog if platform-side carrier error
  4. 4Validate PDF: page count within limit, readable resolution, no password lock
  5. 5Retry or rectify in-session — Jira + PagerDuty if carrier reject with production impact
Result: Fax delivered or clear explanation + next steps for user.
4
Fax · Can't send / credits & format
User blocked: "Insufficient pages," PDF rejected, or cover page error.
0/5
  1. 1Internal admin profile: plan, fax pages/credits, team suspension, sender permissions
  2. 2Check file: PDF only, page limit, file size, no encryption blocking render
  3. 3Cover page: sender name, fax number, and recipient fields complete
  4. 4Stripe vs admin profile vs SQL entitlement row — sync plan/credits or add pages
  5. 5Test send to known-good number — document fix in Zendesk; Jira if platform billing bug
Result: User sending faxes again. Billing/format issue resolved.
